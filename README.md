# Rock_Paper_Scissors_2019
In this RPS game, the user can play against the computer.

Dependencies:
This program has been written in JavaScript and testing was done with cucumber,
other tools used;
NodeJS
Chai
webpack
puppeteer
e2e_training_wheels
babel
yarn command

How to Use tests
$ yarn run cucumber
